Welcome to AfterStorm!



This is a project for Human Computer Interaction (C S 3053) at the University of Oklahoma, taught by Nic Grounds.
This website should work on the Google Chrome browser and on its mobile equivalent.
In order to create a website that is compatible with older computers and cell phones, the layout of the site is very simple and uses as few elements as possible.
If you have any questions, please navigate to the 'Contact' page on the navigation bar above to ask the staff!
Thank you for visiting AfterStorm!
